## 为什么叫 fast 

不是因为它速度快，哈哈

- fast想到什么

1. FAST射电望远镜
2. fastjson  

对很厉害的项目，对它们致敬。 

## 如何引用

1. pom文件加入依赖
```xml
<dependency>
	<groupId>cn.iflyapi.validator</groupId>
	<artifactId>fast-validator</artifactId>
	<version>0.0.1-SNAPSHOT</version>
</dependency>
```

2. 启动类配置

```java
@ComponentScan("cn.iflyapi")
```
